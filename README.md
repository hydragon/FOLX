# FOLX
Our paper ["Extending First-Order Logic for Factual Reasoning over Knowledge Graphs"](https://aclanthology.org/2026.acl-long.1230/) has been published by ACL 2026.

## Dataset Format

This repository contains a dataset for natural language claim verification with corresponding first-order logic (FOL) annotations.

The dataset is stored under the `dataset/` directory and is split into three JSON files:

```text
dataset/
├── train.json
├── val.json
└── test.json
```
